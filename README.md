# Sales-Dashboard
Interactive sales dashboard which provide real time data using power query

📊 Sales Dashboard – Power BI Project
📌 Project Overview

This Sales Dashboard is an interactive Business Intelligence solution developed using Power BI and Power Query. The project transforms raw and messy transactional data into meaningful insights by performing data cleaning, transformation, and visualization.

The dashboard provides a comprehensive analysis of sales performance across multiple dimensions including monthly trends, product performance, payment methods, customer ratings, and geographic distribution.

🛠️ Tools & Technologies Used

Power Query – Data cleaning, filtering, and transformation

Power BI – Data modeling and interactive dashboard creation

DAX – Calculated measures and KPIs

Excel / CSV Dataset – Raw sales data source

🔄 Data Preparation (Power Query)

The raw dataset was processed using Power Query with the following steps:

Removed duplicates and handled missing values

Cleaned and standardized column names

Filtered irrelevant records

Created calculated columns for better segmentation

Transformed data types for accurate analysis

Structured data for efficient modeling

📈 Dashboard Features

The dashboard includes:

🔹 KPI Cards

Total Sales

Total Quantity

Total Transactions

Average Price

🔹 Visualizations

📅 Monthly Sales Trend

📦 Sales by Product Model

💳 Transactions by Payment Method

⭐ Customer Ratings Distribution

🌍 Sales by Location (Map View)

🔹 Interactive Filters (Slicers)

Month-wise Sales Filtering

Payment Method Filter

Brand & Product Model Filter

Users can dynamically select any month using the slicer to analyze sales performance and identify seasonal trends.

📊 Business Insights Generated

Identified peak sales months and seasonal patterns

Analyzed product models contributing highest revenue

Evaluated customer payment preferences

Measured impact of transaction volume on overall sales

Assessed customer satisfaction using rating distribution
